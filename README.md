# Applying the Max-Flow Min-Cut Theorem to Analyze the Network Capacity of the College of Science in Federizo Hall
### Mathematical Modeling (MAT 305) - BSM CS 3A

## Project Overview
This project investigates the capability of the existing network
infrastructure in Federizo Hall to accommodate the total student
population. By representing the system as a flow network, the
Max-Flow Min-Cut theorem is applied to determine the maximum
data throughput and to identify structural limitations affecting
connectivity performance.

##  Repository Structure

- `src/` → main analysis code  
- `data/` → dataset used
- `figures/` → generated graphs
- `tables/` → summarized results  
- `requirements.txt` → dependencies
